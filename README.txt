# Simulador de Examen de Licencia de Conducir

Este proyecto es una aplicación de escritorio con Python y Tkinter para simular exámenes de práctica y exámenes finales tipo test, conectada a una base de datos MySQL.

## Funcionalidades

- Registro y login de usuario
- Simulador de práctica (20 preguntas aleatorias)
- Simulador final (40 preguntas aleatorias, hasta 3 intentos)
- Dashboard con gráficas de rendimiento
- Historial de intentos
